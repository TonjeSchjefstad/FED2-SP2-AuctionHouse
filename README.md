## Getting Started

### Installing

1. Clone the repository:

```bash
git clone https://github.com/TonjeSchjefstad/FED2-SP2-AuctionHouse.git
```

2. Install dependencies:

```
npm install
```

### Running

Start the development server with:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
